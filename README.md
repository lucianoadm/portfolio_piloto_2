📦 Otimização de Almoxarifado: Case Study & Portfolio
Uma aplicação SPA (Single Page Application) de alto nível desenvolvida para apresentar de forma interativa e visual os resultados de um projeto de otimização logística. O projeto utiliza uma estética Modern Organic para equilibrar profissionalismo e modernidade.
🎨 Design & Experiência
A aplicação foi construída sobre a paleta de cores Modern Organic, focada em legibilidade e sofisticação:
Fundo: #FAF9F6 (Off-white)
Acentos: #BC6C25 (Terra Cota) e #283618 (Verde Musgo)
Efeitos: Glassmorphism na navegação e animações fluidas de scroll.
🚀 Funcionalidades Principais
Hero Section Dinâmica: Introdução impactante com estatísticas rápidas e botão de CTA com animação pulse.
3D Flip Cards: Cards interativos para visualização de Problema vs. Solução, girando 180° ao clique.
Dashboard de Performance: Visualização de dados real utilizando Recharts, com KPIs métricos e gráficos de área comparativos.
Timeline do Projeto: Linha do tempo vertical detalhando as fases de Diagnóstico, Arquitetura e Resultados.
Navegação Inteligente: Navbar com efeito blur e smooth scroll para uma experiência de leitura contínua.
🛠️ Tecnologias Utilizadas
React.js: Biblioteca base para a interface.
Framer Motion: Responsável pelas animações 3D e transições de estado.
Recharts: Renderização de gráficos de alta performance.
Tailwind CSS: Estilização responsiva e efeitos de vidro (glassmorphism).
Lucide React: Biblioteca de ícones modernos.
📈 Resultados Demonstrados no Case
O dashboard integrado exibe métricas reais alcançadas:
Redução de tempo de picking.
Aumento da precisão de inventário.
Otimização de espaço físico.
